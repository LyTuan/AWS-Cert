RDS

  1. RDS là gì?
    • Amazon Relational Database Service là dịch vụ cho phép tạo, thao tác, scale cơ sở dữ liệu quan hệ trên AWS cloud
    • DB instance là một kiểu môi trường cơ sở dữ liệu riêng biệt trong AWS cloud
    • RDS hỗ trợ các engine sau:
      • Mysql
      • MariaDB
      • PostgresSQL
      • Oracle
      • Microsoft SQL Server
  2. Cách tính tiền cho Amazon RDS
    • Có 1 số cách tính tiền như sau:
      • DB instance hours: Giá tiền tính theo giờ, nhưng hóa đơn sẽ tính theo giây, tối thiểu để tính billing RDS là 10 phút
      • Storage(GiB/tháng)
      • I/O request
      • Provisioned IOPS (1 triệu request mỗi tháng): Áp dụng cho kiểu RDS magetic
      • Backup storage
      • Data transfer
