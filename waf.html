IAM

  1. WAF là gì?

    • WAF (Web application firewall) là tưởng lửa ứng dụng web chống lại các cuộc tấn công vào ứng dụng web và các con bots
    • Cho phép người dùng cấu hình các Rule

  2. Tường lửa làm được gì?

    • Cho phép tất cả các request đi qua ngoại trừ 1 số request đặc biệt
    • Chăn tất cả các request ngoại trừ 1 số request đặc biệt được phép đi qua
    • Đếm số lượng các request có các đặc điểm khớp với 1 số thuộc tính đặc biệt

    Tường lửa web còn chống lại 1 số loại tấn công phổ biến như:
    • Chặn tất cả các truy cập độc hại: SQL Injection (SQLi), Cross-site scripting (XSS), Remote file inclusion(RFI), DDoS attack on level 7, bad bot dựa vào rule
    • Lọc các truy cập: Đánh giá dựa vào IP, Matching bằng string hoặc regex,
    • Giám sát: Đưa ra logs và các cảnh báo độ đo Amazon Cloud watch
