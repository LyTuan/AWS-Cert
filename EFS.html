EFS


  • EFS là gì?
    • EFS là service hỗ trợ chia sẻ file thông qua NFS (Network file system) trên nhiều EC2 khác nhau. Điều này cũng có nghĩa EFS có thể hoạt động trên nhiều EC2 khác nhau và nằm trên nhiều AZ khác nhau
    • EFS mang lại tính bổ trợ rất cao đối với EBS vì EBS bị khóa trên 1 AZ còn EFS sẽ giúp chúng ta share data trên nhiều AZ khác nhau
    • Chi phí sử dụng EFS khá cao = gp2 * . Chi phí này phụ thuộc vào lượng sử dụng, sử dụng đến đâu thì trả đến đấy
    • EFS sử dụng giao thức NFSv4.1
    • EFS sử dụng tương thích với các AMI base trên linux
    • Performance mode:
      • General purpose: Phục vụ các app thông thường cần chia sẻ file hệ thống với nhiều file nhỏ
      • Max I/O: Phục vụ các big data hay media processing cần through put cao và lacenty tốt hơn
    • Lưu trữ:
      • Standard: các file thường xuyên được truy cập
      • EFS-IA: Các file không thường xuyên được truy cập sau N ngày. N=30. Mục đích lưu trữ dạng này chi phí thấp hơn. Khi file được truy cập thì ngay lập tức trở thành standard
